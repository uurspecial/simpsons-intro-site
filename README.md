# 辛普森家庭網站

這是辛普森一家人的日常。
一家人坐在熟悉的沙發上看電視，吵吵鬧鬧。廚房裡很乾淨，是Marge好不容易整理的...

## 網頁設計技術

本專案使用以下技術構建：

- `HTML`：建立網頁結構和內容，定義畫面中的人物、場景與互動區塊。
- `Flask`：作為後端框架，提供網頁路由、模板渲染、靜態資源服務及簡易伺服器環境。
- `JavaScript`：控制互動效果、影片播放、鈴聲觸發、貓咪移動，以及人物介紹等動態行為。
- `CSS`：設計版型與視覺風格，讓畫面更貼近卡通場景，並改善使用者的閱讀與操作體驗。

### 詳細解說

- `HTML` 負責定義場景區塊，例如客廳、廚房、人物卡片與互動按鈕。
- `Flask` 在 `simpsonapp.py` 中建立路由，將 `templates/` 中的 HTML 模板與 `static/` 下的 CSS、JavaScript、影像、音效等資源結合。
- `JavaScript` 將靜態畫面變成互動體驗，包含：
  - 播放影片與音效
  - 鈴聲觸發
  - 貓咪移動動畫
  - 顯示人物介紹面板
  - 讓使用者感覺自己就在辛普森家的場景裡
- `CSS` 提供色彩、排版、動畫效果以及元件樣式，讓網站視覺更有活力。

## 互動特色

網站包含以下互動元素：

- 影片播放
- 鈴聲效果
- 貓咪移動動畫
- 人物介紹卡片
- 讓使用者沉浸在情境中的趣味互動

## 專案結構

- `simpsonapp.py`：Flask 應用主檔，負責啟動伺服器與路由管理。
- `templates/`：Flask 模板資料夾，包含主頁與場景頁面的 HTML 模板。
- `static/`：靜態資源資料夾，包含：
  - `styles.css`：網頁樣式
  - `scripts/script.js`：互動行為與事件控制
  - `audio/`：鈴聲與音效
  - `images/`：人物與場景圖片
  - `videos/`：影片素材

## 啟動方式

1. 安裝 Python 與 Flask。
2. 在專案目錄下執行：

```bash
python simpsonapp.py
```

3. 開啟瀏覽器並前往 `http://127.0.0.1:5000`。
